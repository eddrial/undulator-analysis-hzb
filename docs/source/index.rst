Welcome to HZB Undulator Analysis's documentation!
==================================================

**HZB Undulator Analysis** is a Python Library that interfaces the 
magnetic measurement tools of Helmholtz Zentrum Berlin to a self describing HDF5 format,
and further allows the processing, post processing, and reporting of such measurements.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project.

.. note::
    This project is under active development
